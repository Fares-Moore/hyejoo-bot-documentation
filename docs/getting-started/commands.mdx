# Interacting with the bot 

## Commands
To interact with the bot, the use of `/`commands are used.

### Common Commands 
These commands that are commonly used will be elaborated in other parts of the documentation:
- `/cards `
- `/cooldown `
- `/claim cards` and `/level claim`
- `/work `
- `/train `
- `/forge `
- `/draw `
