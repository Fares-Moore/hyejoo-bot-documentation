# Seasonal Events 🎄🎃🌸

Seasonal events are limited-time **promo collections** that pop up during holidays and special seasons. These collections have their own cards that you can claim using the event currency alongside your regular claims!

## How seasonal events work

- During a seasonal event, an **active promo** is running in the bot.
- When you claim regular cards using `/claim cards`, you also earn **event currency** equal to **25% of your claim cost**.
- Use `/claim cards promo:true` to spend your event currency on promo cards from the active event collection.
- Every **30th regular claim** during a boost event guarantees a special promo card drop! 🎉
- Promo collections have an expiration date — once they expire, cards can no longer be claimed until the event returns.

### Event currency

Each event has its own temporary currency. You can check your balance with `/balance`.

| Action | Currency earned |
| --- | --- | --- |
| Regular card claim | +25% of claim cost as event currency |
| Claiming event cards | Spends event currency |
| Event achievements | Bonus event currency |

## Commands for events

| Command | What it does |
| --- | --- |
| `/claim cards boostid:` | Claim event cards using an active boost ID |
| `/claim cards count:N` | Claim multiple cards at once (max 10) |
| `/balance` | Check your event currency balance |
| `/daily` | Reset claim cost and get more currency |

## Quick tips

- **Stack up during events** — Claim as many regular cards as you can during an event to earn event currency, then use it to grab promo cards.
- **30-claim guarantee** — Every 30th regular claim during an active boost gives a guaranteed boost card drop. Time your claims!
- **Don't let it expire** — Keep an eye on the event end date. Unused event currency disappears when the event ends.
- **Check achievements** — The event achievements (first promo, 5 promo, 10 promo) give bonus currency and lemons!
