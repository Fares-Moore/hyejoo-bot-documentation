# Community Events 🌐

Beyond the bot's automated events, server communities and the Hyejoo team also organize fun community-driven events! These are typically run by server admins or bot moderation staff.

## Types of community events

### Card Giveaways 🎁

Server admins and Ko-Fi supporters sometimes run card giveaways where players can win special cards! These giveaways may use:
- Random draws from participants
- Trivia or quiz competitions
- Activity-based rewards

### Community Games 🎮

Occasionally, the community organizes mini-games and challenges in the bot's Discord server. Past events have included:
- **Trivia nights** — K-Pop trivia with card or currency prizes
- **Card design contests** — Submit card concepts for a chance to have them added
- **Collection races** — First to complete a specific collection during an event wins a prize

### Guild Events 🏘️

Some guilds organize their own internal events for members:
- Guild-exclusive auctions
- Internal leaderboard competitions
- Collaborative collection goals

### Special Promotions

Sometimes the bot team runs unique promotions:
- **Ko-Fi supporter perks** — Monthly card draws from exclusive collections
- **Milestone celebrations** — When the bot reaches a milestone (e.g., number of cards, users, or collections), special giveaways may happen
- **Holiday surprises** — Unexpected mini-events during holidays beyond the regular seasonal promos

## How to stay informed

- 📢 **Check `/announcement`** — The latest bot news and announcements
- 💬 **Join the Bot Discord** — Link available via `/invite`
- 📋 **Follow `/todo`** — Shows a checklist of things you can do today, including any active events
- 🎫 **Check `/kofi`** — For Ko-Fi supporters, shows available rewards and promotions

## Commands for community interaction

| Command | What it does |
| --- | --- |
| `/announcement` | View the latest bot announcement |
| `/invite` | Get the invite link to the bot's Discord server |
| `/report` | Open the report/suggestion form |
| `/kofi` | Show Ko-Fi support info and premium perks |
| `/vote` | Vote for the bot (12h cooldown, gives rewards) |

## Quick tips

- **Stay active in the community** — Many events are announced first in the Discord server before anywhere else.
- **Use `/announcement` regularly** — The latest updates, events, and promotions are always posted there.
- **Events aren't just for cards** — Community events are a great way to meet other collectors and have fun!
- **Suggest your own ideas** — Use `/report` to suggest new event ideas to the bot team.
